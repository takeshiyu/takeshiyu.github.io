Adding Apline.js to Laravel using Laravel Mix

作者
  • 名稱
    Takeshi Yu
    Twitter
    @easternsoul1

Alpine.js 是最近在社群裡常出現的一個輕量的 JavaScript 框架，它與 Laravel Livewire 都是由 Caleb Porzio 所開發出來的，兩者互相搭配後，可以與前端頁面達到交互作用的效果，對於不是很熟悉前端技術的我而言，是個很不錯的選擇！

雖然說直接透過使用 script tag 來安裝使用 Apline.js 也是可以的，官方文件 也有提到，但是如果能把需要用到的 JavaScript 以及 CSS 透過 Webpack 加入到 Laravel 裡就好了，這樣可以透過 Compiling Assets (Mix) 將多個檔案都壓縮在一起，加快載入的速度，而實際使用時，只需要加上 app.js 以及 app.css 就好，也比較乾淨。

安裝 Apline.js #

依照 官方文件 的描述，在專案目錄裡執行 npm 來安裝：

npm install alpinejs

安裝成功後，需要編輯 resources/js/app.js 載入 Alpine.js:

import './bootstrap';

// 加入下列程式碼...
import Alpine from 'alpinejs';
window.Alpine = Alpine;
Alpine.start();

完成後，接著執行：

npm run dev

執行完成後，可以打開 public/js/app.js 查看是否有成功加入 Alpine.js。

測試 #

調整一下預設的 resources/views/welcome.blade.php:

<!DOCTYPE html>
<html lang="{{ str_replace('_', '-', app()->getLocale()) }}">
    <head>
        <meta charset="utf-8">
        <meta name="viewport" content="width=device-width, initial-scale=1">
        <title>Alpine.js Demo</title>
        <link rel="stylesheet" href="{{ asset('css/app.css') }}">
        <script type="text/javascript" src="{{ asset('js/app.js') }}" defer></script>
    </head>
    <body class="antialiased">
        <div x-data="{ show: true }">
            <button @click="show = !show">Hide</button>
            <h1 x-show="show">Hello World! Alpine.js</h1>
        </div>
    </body>
</html>

因為 show 預設為 true，因此打開頁面後就可以直接看到 Hello World! Alpine.js 的訊息，若點擊按鈕 Hide，則 show 會被設定為 false，訊息就會被隱藏了。

Alpine.js 成功安裝！
